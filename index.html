Hanzi Hero: Empowering Kids to Master Chinese Characters

This project aims to build an image recognition model for handwritten Chinese characters and deploy it to Streamlit, so that we can assist students and non-Chinese learners better with their learning of the Chinese language. The first repository provides all the processes and codes needed to build the image recognition model, whereas the second repository focuses on deploying the trained model to a user interface using Streamlit Cloud. The user interface can be accessed via this link: https://hanzi-hero.streamlit.app/.

In A Heartbeat: Prediction of Heart Disease Risk for Early Detection

Heart disease remains the leading cause of death in the United States, a concerning statistic that has persisted for over a century. This project addresses this critical public health issue by developing a heart disease risk prediction model. Our primary objective is to raise awareness of this prevalent condition and encourage early detection, a crucial factor in mitigating risk. We utilized data from the 2022 US Behavioral Risk Factor Surveillance System (BRFSS), implementing rigorous data cleaning, pre-processing, and imputation techniques to address missing values. An interactive Tableau dashboard is available for further data exploration.
